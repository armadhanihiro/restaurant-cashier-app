## 📌 Project Overview
This project is a restaurant cashier application built using Java. It is designed to manage restaurant operations such as customer data, food and drink menus, and order processing through a graphical user interface.

The application provides a simple desktop-based system for handling restaurant transactions efficiently.

---

## 🧾 Project Description
The system includes several main components such as login, menu management, customer data, cashier interface, and order handling. It is structured using multiple Java classes to represent restaurant entities and application views.

This project demonstrates the implementation of object-oriented programming concepts and desktop GUI development using Java.

---

## ⚙️ Features
- Login interface
- Cashier dashboard
- Customer data management
- Food and drink menu management
- Order processing
- Desktop-based graphical user interface

## ▶️ How to Run
1. Open the project in a Java IDE such as NetBeans or IntelliJ IDEA
2. Compile the project
3. Run `Driver.java` as the main entry point
4. Use the application through the provided GUI
